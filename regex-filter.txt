# ============================================
# HINWEIS ZUR VERWENDUNG:
# ============================================
# Diese Regex-Einträge können NICHT als normale Blocklist importiert werden!
# 
# So fügst du sie hinzu:
# 1. Pi-hole Admin Interface öffnen
# 2. Zu "Filters" → "Regex Filters" navigieren
# 3. Jeden Regex-Eintrag einzeln hinzufügen
# 4. Als "Blacklist" markieren
# 
# Die Regex-Syntax (\.|^) matched sowohl Subdomains als auch die Root-Domain
# Beispiel: (\.|^)baidu\.com$ blockiert sowohl baidu.com als auch www.baidu.com
